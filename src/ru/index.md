## Что еще добавить на лэндинг?

Для создания полноценного продающего лэндинга рекомендую добавить следующие блоки под секцией Features:

### 1. Видео-демонстрация (Showcase)
**Что воткнуть:** Фоновое зацикленное видео или качественный GIF в высоком разрешении, показывающий интерфейс FastCat в действии.
- **Интерактив:** Наложение прозрачных меток, которые пульсируют и при наведении показывают подсказки (например, "Таймлайн с магнитным захватом", "Панель ИИ-инструментов").
- **Картинка:** Скриншот редактора с открытым проектом, где много слоев и эффектов.

### 2. Сравнение "До и После" (AI Magic)
**Что воткнуть:** Интерактивный слайдер (ползунок), разделяющий экран пополам.
- **Слева:** Исходное видео с шумом, тусклыми цветами и фоном.
- **Справа:** Видео после обработки FastCat (чистый звук, цветокор, удаленный фон, добавленные субтитры).
- **Анимация:** Плавное движение ползунка автоматически при скролле.

### 3. Карусель ключевых фишек ИИ
**Что воткнуть:** Слайдер с карточками, описывающими детально каждую ИИ-функцию.
- **Картинки:** Короткие видео-лупы (5-10 сек) для каждой функции (процесс удаления фона, процесс появления субтитров под музыку).
- **Интерактив:** При клике на карточку открывается мини-плеер с примером работы именно этой функции.

### 4. Блок "Для кого FastCat?"
**Что воткнуть:** Сетка с иконками и коротким текстом для разных сегментов пользователей:
- **Блогеры:** "Быстрый монтаж Reels и TikTok за 5 минут".
- **Педагоги:** "Создание обучающих курсов с авто-субтитрами и озвучкой".
- **SMM:** "Готовые пресеты для соцсетей и быстрая адаптация форматов".

### 5. Пошаговый процесс (How it works)
**Что воткнуть:** Горизонтальная или вертикальная линия прогресса с шагами:
1. **Импорт:** "Просто перетащите файлы в окно".
2. **AI-Магия:** "Нажмите одну кнопку для улучшения".
3. **Финальные штрихи:** "Добавьте переходы и текст".
4. **Экспорт:** "Получите готовое видео в нужном формате".

### 6. Секция призыва к действию (CTA)
**Что воткнуть:** Яркий блок внизу страницы с большой кнопкой "Скачать FastCat бесплатно" и списком поддерживаемых платформ (Windows, macOS, Linux).
- **Анимация:** Плавное появление (fade-in) кнопки при достижении низа страницы.
