# Горячие клавиши (Shortcuts)

Использование горячих клавиш значительно ускоряет процесс монтажа. FastCat использует стандартные для индустрии комбинации клавиш.

## Управление проектом
- **Ctrl + N**: Создать проект
- **Ctrl + O**: Открыть проект
- **Ctrl + S**: Сохранить проект
- **Ctrl + Z / Ctrl + Y**: Отмена / Повтор действия

## Редактирование на таймлайне
- **V**: Инструмент выделения (Selection)
- **C**: Инструмент обрезки (Razor / Cut)
- **Del / Backspace**: Удалить клип
- **Shift + Del**: Удалить со сдвигом (Ripple Delete)
- **Ctrl + D**: Добавить переход по умолчанию

## Навигация и Предпросмотр
- **Пробел**: Старт / Стоп воспроизведения
- **J / K / L**: Назад / Стоп / Вперед (со скоростью)
- **I / O**: Установить метки Вход / Выход (In / Out)
- **Стрелки влево/вправо**: Перемещение на 1 кадр
- **Shift + стрелки**: Перемещение на 1 секунду
