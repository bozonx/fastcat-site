# Мобильная версия

FastCat на мобильных устройствах — это полноценный видеоредактор в вашем кармане. Интерфейс адаптирован для управления пальцами и работы с вертикальными видео.

## Сенсорное управление (Жесты)
- **Тап (Нажатие)**: Выделение клипа или выбор инструмента.
- **Долгое нажатие**: Переход в режим перемещения клипа.
- **Двупальцевый жест (Pinch)**: Масштабирование таймлайна (раздвигание — зум, сведение — отдаление).
- **Свайп по таймлайну**: Быстрая прокрутка видео.
- **Тап двумя пальцами**: Быстрая отмена действия (Undo).

## Особенности интерфейса
- **Вертикальный режим**: По умолчанию проекты создаются в формате 9:16.
- **Скрываемые панели**: Чтобы освободить место для видео, Инспектор и Библиотека файлов прячутся в нижней части экрана и открываются по клику.
- **Компактный таймлайн**: Основные инструменты (ножницы, удаление) находятся всегда под рукой, прямо над таймлайном.

## Специфика мобильного монтажа
- **Быстрая обрезка**: Используйте специальные "ручки" по краям клипа для точной подгонки под музыку.
- **Запись закадрового голоса**: Нажмите на иконку микрофона прямо на таймлайне, чтобы начать запись озвучки.
- **Предпросмотр**: Нажмите на экран предпросмотра, чтобы развернуть видео на весь экран и оценить результат.

## Оптимизация и Батарея
- **Экономия энергии**: Мобильная версия автоматически снижает качество предпросмотра, если заряд батареи низкий.
- **Кэширование**: Для плавной работы рекомендуем очищать временные файлы в настройках, если на телефоне заканчивается память.
- **Экспорт**: Видео сохраняется напрямую в галерею вашего устройства или сразу отправляется в нужное приложение (Instagram, TikTok, Telegram).
