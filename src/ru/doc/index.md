# Документация функций

Здесь приведен полный перечень возможностей видеоредактора FastCat.

## Управление проектами
- Создание и сохранение проектов
- Экспорт в формат OTIO (OpenTimelineIO)
- Настройка параметров проекта (разрешение, FPS)
- Метаданные и тегирование

## Редактирование на таймлайне
- Инструменты выделения (Selection)
- Обрезка и склейка (Trim/Split)
- Группировка клипов
- Управление слоями и треками
- Система Undo/Redo (История действий)

## Работа с медиафайлами
- Импорт через Virtual File System (VFS)
- Генерация прокси-файлов
- Кэширование медиа-данных
- Автоматическое распознавание метаданных (Exifr)

## Визуальные эффекты (Video Effects)
- Цветокоррекция и фильтры (PixiJS Filters)
- Маскирование и обтравка
- Наложение текста и титров
- Трансформация (Scale, Rotation, Position)

## Аудио инструменты
- Регулировка громкости и панорамирование
- Визуализация аудиоволны (Wavesurfer)
- Удаление шума и нормализация
- Наложение фоновой музыки

## Переходы (Transitions)
- Наплыв (Dissolve)
- Сдвиг (Slide)
- Вытеснение (Wipe)
- Кастомные переходы (Shader-based)

## Инструменты ИИ
- ИИ-ассистент для монтажа
- Автоматические субтитры (Whisper/Transformers.js)
- Генерация контента по описанию
- Умное выделение объектов

## Рендеринг и Экспорт
- Настройка видеокодеков
- Очередь рендеринга
- Параметры экспорта аудио
- Интеграция с FFmpeg
