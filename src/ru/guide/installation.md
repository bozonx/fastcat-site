# Установка и Запуск

Вы можете запустить FastCat несколькими способами. Выберите тот, который вам удобнее.

## Способ 1: С помощью Docker (Самый простой)
Этот способ не требует установки Node.js и настройки окружения. Вам нужен только установленный [Docker](https://www.docker.com/products/docker-desktop/).

1. Скачайте проект или склонируйте репозиторий.
2. Откройте терминал в папке проекта.
3. Запустите команду:
   ```bash
   docker-compose up -d
   ```
4. Откройте браузер по адресу `http://localhost:3000`.

## Способ 2: С помощью Node.js (Для работы с кодом)
Если вы хотите запускать редактор локально и иметь доступ к файлам конфигурации.

### 1. Подготовка
- Установите **Node.js** (версия 18 или выше) с официального сайта [nodejs.org](https://nodejs.org/).
- Скачайте репозиторий с кодом (кнопка "Download ZIP" на GitHub или через `git clone`).

### 2. Установка зависимостей
Откройте терминал (командную строку) в папке с проектом и введите:
```bash
npm install
```
*Эта команда скачает все необходимые компоненты для работы редактора.*

### 3. Запуск
Для обычного использования запустите:
```bash
npm run start:prod
```
После этого редактор будет доступен в браузере по адресу `http://localhost:5173` (или другому адресу, указанному в терминале).

## Требования к браузеру
Для работы нейросетей и быстрой обработки видео FastCat использует технологию **WebGPU**. Мы рекомендуем использовать последние версии:
- Google Chrome
- Microsoft Edge
- Браузеры на базе Chromium
