## Самостоятельный запуск в браузере (использование OPFS)

Для самостоятельного запуска FastCat используется Docker. Это самый простой и надежный способ, который не требует сложной настройки локального окружения.

### Установка с помощью Docker

Вам потребуется только установленный [Docker](https://www.docker.com/products/docker-desktop/).

1. Скачайте проект или склонируйте репозиторий `https://github.com/bozonx/fastcat.git`.
2. Откройте терминал в папке проекта.
3. Запустите команду:
   ```bash
   docker compose up -d
   ```
4. Откройте браузер по адресу `http://localhost:3000`.

### Требования к браузеру (OPFS)

**Важно:** Открывать пользовательский интерфейс (UI) FastCat можно только в браузерах, которые поддерживают технологию **OPFS** (Origin Private File System). 

На данный момент эту технологию полноценно поддерживают браузеры на базе **Chromium**. Основные из них:
- **Google Chrome**
- **Chromium**
- **Microsoft Edge**
- **Opera**
- **Vivaldi**
- **Brave**

Убедитесь, что вы используете последнюю версию браузера.

### Ручной запуск

Для ручного запуска (например, с использованием Node.js для работы с кодом), пожалуйста, обратитесь к инструкциям в файле [README.md](https://github.com/bozonx/fastcat.git) в репозитории проекта.
